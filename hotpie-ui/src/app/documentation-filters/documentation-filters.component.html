To Include or not to Include...

If you don't need some of the data, it is just simpler to discard that from your analysis. The flip side to this is also valid. If you are interested in only a single type of data, then why have the rest there.
We offer two options

Exclude

Provide us the pattern for the filter value and the filter value itself, and we will make the data unit vanish.

Include

Provide us the pattern for the filter value and the filter value itself, and we will make sure that we include only those data units for you.

Note: What we need to keep in mind here, is that the procesing in done in a pipeline. This means that that if you have added two filters, then the result of the first filter will be taken as the input for the second.
