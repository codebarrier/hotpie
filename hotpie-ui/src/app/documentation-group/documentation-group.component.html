Grouping Up

Often in data, there are some data units that belong together and there are others that do not. Here we attempt to group those data units based on certain predefined creteria, that we can establish in the profile.
We use regex again to identify the parts of the data units that we can use to group them together.
We realize that there are often times where we would have to group data units more than once and we therefore being multi level grouping to you.
